# Resend

Resend email API emulation with email sending, domain management, API keys, audiences, contacts, and a local inbox for captured messages.

## Emails

- `POST /emails` тАФ send single email
- `POST /emails/batch` тАФ send up to 100 emails
- `GET /emails` тАФ list sent emails
- `GET /emails/:id` тАФ get email
- `POST /emails/:id/cancel` тАФ cancel scheduled email

## Domains

- `POST /domains` тАФ create domain
- `GET /domains` тАФ list domains
- `GET /domains/:id` тАФ get domain
- `DELETE /domains/:id` тАФ remove domain
- `POST /domains/:id/verify` тАФ trigger domain verification

## API Keys

- `POST /api-keys` тАФ create API key
- `GET /api-keys` тАФ list API keys
- `DELETE /api-keys/:id` тАФ delete API key

## Audiences & Contacts

- `POST /audiences` тАФ create audience
- `GET /audiences` тАФ list audiences
- `DELETE /audiences/:id` тАФ delete audience
- `POST /audiences/:audience_id/contacts` тАФ add contact
- `GET /audiences/:audience_id/contacts` тАФ list contacts
- `DELETE /audiences/:audience_id/contacts/:id` тАФ delete contact

## Inbox

- `GET /inbox` тАФ list captured emails
- `GET /inbox/:id` тАФ view captured email
